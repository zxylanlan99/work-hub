## 一、全局设计规范

### 1.1 主题

- 统一白色/浅色主题（light mode）
- CSS 变量体系：`--primary: #6366f1` / `--primary-dark: #4f46e5` / `--primary-light: #eef2ff`
- 成功色：`--success: #22c55e` / 警告色：`--warning: #f59e0b` / 危险色：`--danger: #ef4444` / 信息色：`--info: #3b82f6`
- 灰色体系：`--gray-50 ~ --gray-900` 完整色阶

### 1.2 布局

- 侧边栏 220px（`--sidebar-bg: #f9fafb`）+ 主区域
- 顶栏 56px（白色背景 + 底部分割线）
- 内容区 `background: #f0f2f5`，内边距 24px

### 1.3 左侧菜单（7 项统一）

```
📊 首页 → 🎯 学习计划 → 📡 资讯 → 📚 知识库 → 💬 AI对话 → 🔁 复习计划 → ✍️ 知识沉淀
```

### 1.4 组件风格

- 按钮：`.btn-primary` / `.btn-secondary` / `.btn-ghost` 三色体系
- 弹窗：`.modal-overlay` 半透明遮罩 + `.modal` 白色圆角卡片 + `modalIn` 动画
- Toast：右上角固定定位 + 滑入动画 + 3 秒自动消失
- 子 Tab：`.sub-tabs` 灰色胶囊容器 + `.sub-tab.active` 白色高亮

## 二、各模块设计方案

### 模块 1：首页（`prototype-首页模块.html`）

**定位**：总览仪表盘，聚合各模块关键信息。

**页面结构**：

| 区域 | 内容 | 来源 PRD 节点 |
|------|------|--------------|
| 暖身卡片 | 昨日回顾 1 句 + 快问快答 1 题 + 继续/跳过 | 1C 暖身引擎 |
| 四宫格快捷入口 | 学习计划/今日复习/待处理资讯/输出草稿 四项关键数字 | 各模块聚合 |
| 智能续接 | 上次学习位置 + 预计剩余时间 + 一键跳转 | 1D 智能续接 |
| 学习日历热力图 | 近 2 周复习日历（颜色深浅 4 级） | 可视化 |
| 本周趋势 | 复习完成率 72% + 环比 +8% + 连续 7 天 | 统计聚合 |

**特殊交互**：
- 快问快答：点击"查看答案"→ 展开绿色答案标签
- 暖身可跳过：点击"跳过暖身"隐藏整条暖身卡
- 四宫格卡片可点击跳转对应模块（Toast 提示）

**定位**：智能交互中枢 — 模型路由 + 对话管理 + 知识连接 + 费用看板。

**页面结构**：

| 区域 | 内容 |

**弹窗**：

| 弹窗 | PRD 节点 |
|------|----------|

| 收藏入库 | 复用入库弹窗 + 自动填标题/摘要/来源 |

## 四、模块间数据流

```
首页（总览仪表盘）
  ├─→ 学习计划（AI 规划目标路径）
  ├─→ 资讯（AI 筛选+入库）→ 知识库（存储+分类）
  ├─→ AI对话（智能路由+知识检索）→ 知识库（收藏入库）
  ├─→ 知识库 → 复习计划（自动生成复习卡片）
  └─→ 知识库 → 知识沉淀（素材匹配+输出转化）
        └─→ 复习计划（发布后生成复习卡片）
```
