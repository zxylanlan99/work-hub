## 四、卡片系统规范

### 4.1 卡片层级原则

**精简原则：最多3层卡片嵌套**

```
Level 3: 最外层卡片（.panel/.goal-card/.overview-card）
    ↓
Level 4: 卡片内部区域（.panel-header/.panel-body）
    ↓
Level 5: 子组件（使用div + class，不再嵌套卡片）
```

### 4.2 卡片内间距规范

| 卡片类型 | 内间距 | 使用场景 |
|---------|---------------|---------|
| **最外层卡片** | 20px | `.goal-card`、`.detail-hero` |
| **标准卡片** | 16px | `.panel-body`、`.overview-card` |
| **紧凑卡片** | 12px | `.milestone-preview`、弹窗内嵌卡片 |
| **卡片头部** | 14px 16px | `.panel-header`、`.milestones-header` |
| **卡片底部** | 16px 24px | `.modal-footer` |

### 4.3 卡片间距规范

| 间距类型 | 间距值 | 使用场景 |
|---------|--------|---------|
| **最外层卡片间距** | 16px ~ 20px | `.goals-grid`、`.content`内卡片之间 |
| **概览卡片间距** | 16px | `.overview-grid` |
| **统计卡片间距** | 14px | `.stat-cards` |
| **卡片内部间距** | 10px ~ 16px | 卡片内子元素之间 |
| **紧凑内部间距** | 6px ~ 10px | `.diag-item`、`.task-item` |
| **详情页双栏间距** | 20px | `.detail-layout` |
| **弹窗内间距** | 16px | 弹窗内卡片之间 |

### 4.4 卡片宽度规范

| 卡片类型 | 宽度设置 | 最小宽度 | 说明 |
|---------|---------|---------|------|
| **网格卡片** | `minmax(340px, 1fr)` | 340px | `.goals-grid`自适应网格 |
| **概览卡片** | `repeat(4, 1fr)` | 无固定 | `.overview-grid`四等分 |

### 4.5 卡片内容溢出处理

| 溢出类型 | 处理方式 | CSS代码 |
|---------|---------|---------|
| **标题过长** | 单行截断+省略号 | `overflow:hidden;text-overflow:ellipsis;white-space:nowrap` |
| **描述过长** | 多行截断(2-3行) | `overflow:hidden;line-height:1.5;max-height:3.75em` |
| **列表过长** | 滚动容器 | `max-height:300px;overflow-y:auto` |
| **弹窗内容过长** | body滚动 | `max-height:70vh;overflow-y:auto` |

### 4.6 文字截断CSS

```css
/* 单行截断 */

  overflow: hidden;

/* 两行截断 */

  -webkit-box-orient: vertical;
```

## 五、弹窗规范

### 5.1 弹窗尺寸

| 尺寸类型 | CSS类名 | 最大宽度 | 适用场景 |
|---------|---------|---------|---------|
| **小弹窗** | `.modal-sm` | 440px | 确认删除、简单提示 |
| **中弹窗** | `.modal-md` | 580px | 新建任务、添加内容 |
| **大弹窗** | `.modal-lg` | 720px | AI诊断报告、详情编辑 |
| **超大弹窗** | `.modal-xl` | 880px | 对比模式、分屏编辑 |

### 5.2 弹窗结构

```
.modal-overlay（遮罩层，z-index:100）
    └── .modal（弹窗容器）
            ├── .modal-header（头部）
            │       ├── .modal-title（标题，16px/600）
            │       ├── .modal-subtitle（副标题，12.5px/400）
            │       └── .modal-close（关闭按钮，20px）
            │
            ├── .modal-body（主体，max-height:70vh）
            │       └── [弹窗内容]
            │
            └── .modal-footer（底部）
                    └── [操作按钮组]
```

### 5.3 弹窗动画

| 动画名称 | 效果 | 时长 |
|---------|------|------|
| **进入动画** | 从上滑入 | 0.25s |
| **遮罩模糊** | 背景模糊4px | - |

```css
@keyframes slideIn {
  from { opacity: 0; transform: translateY(-12px); }
  to { opacity: 1; transform: translateY(0); }
}
```

### 5.4 弹窗使用规范

| 规范项 | 说明 |
|-------|------|
| **打开方式** | 给`.modal-overlay`添加`.show`类 |
| **关闭方式** | 移除`.show`类 |
| **点击遮罩关闭** | 可选实现，需绑定点击事件 |
| **ESC键关闭** | 可选实现，需绑定键盘事件 |
| **内容滚动** | `.modal-body`设置`max-height:70vh`+`overflow-y:auto` |
| **按钮排列** | `.modal-footer`使用`justify-content:flex-end` |
