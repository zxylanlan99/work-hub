## 九、实现注意事项

1. **视图切换**：列表页和详情页通过切换 `.content` 区块实现，不是 URL 路由
2. **阅读进度**：使用 `onscroll` 事件实时更新进度条宽度百分比
3. **批量选择**：仅在 AI推荐清单 tab 启用，`.cb` 自定义 checkbox + `.batch-bar` 浮动操作栏
4. **分类树折叠**：使用 `▶` 图标 + CSS transform 旋转，`.cat-hidden` 控制子节点显示
5. **Markdown 预览**：编辑/预览 Tab 切换，右侧实时渲染 Markdown（使用前端库如 marked.js）
6. **搜索节流**：搜索输入使用 debounce（300ms）避免频繁请求
7. **空状态**：分类树和列表均需处理空状态
8. **Ctrl+K 快捷键**：`document.addEventListener('keydown')` 全局监听
