## 一、功能需求分析

### 1.1 核心功能点

| 功能模块 | 功能描述 | 对应原型区域 |
|---------|---------|-------------|
| **今日复习** | 展示待复习卡片，支持翻转查看答案、评分 | 主区域卡片 |
| **复习历史** | 热力图展示近3个月复习情况 + 复习记录列表 | 历史页面 |
| **统计总览** | 掌握度分布、遗忘曲线、薄弱主题 | 统计页面 |
| **串联复习** | 关联知识建议，可加入今日复习 | 侧边栏/弹窗 |
| **遗忘预警** | 高风险卡片提醒 | 顶栏按钮/弹窗 |

### 1.2 页面结构

```
.review-page
    ├── .sub-tabs (子标签：今日复习/复习历史/统计总览)
    │
    ├── #page-today (今日复习页)
    │       ├── .overview-grid (概览卡片：应复习/已完成/剩余/连续天数)
    │       ├── .alert-banner (提醒条：逾期提醒、串联建议)
    │       └── .review-layout (双栏布局)
    │               ├── .review-main (复习卡片区域)
    │               │       ├── .flashcard-perspective
    │               │       │       └── .flashcard (翻转卡片)
    │               │       └── .card-nav (卡片导航)
    │               └── .review-sidebar (侧边栏面板)
    │                       ├── 知识点来源
    │                       ├── 掌握度趋势
    │                       └── 关联知识
    │
    ├── #page-history (复习历史页)
    │       ├── .history-layout (双栏布局)
    │       │       ├── .heatmap-panel (热力图)
    │       │       └── .history-list (复习记录)
    │
    └── #page-stats (统计总览页)
            ├── .overview-grid (统计概览)
            └── .stats-grid (三个统计面板)
                    ├── 掌握度分布
                    ├── 遗忘曲线
                    └── 薄弱主题
```

## 二、布局与样式规范

### 2.1 布局结构

| 区域 | 宽度 | 样式特征 |
|------|------|---------|
| 主内容区 | 自适应 | 白色背景、圆角 12px、阴影 |
| 复习布局 | grid (1fr 280px) | 主区域 + 侧边栏 |
| 侧边栏 | 280px 固定 | 白色背景、圆角 12px |
| 翻转卡片 | perspective: 1000px | 3D翻转效果 |

### 2.2 颜色规范

| 元素 | CSS变量 | 说明 |
|------|---------|------|
| 主背景 | `--content-bg` | 页面背景 |
| 卡片背景 | `--bg` (#ffffff) | 卡片背景 |
| 逾期标记 | `--danger` (#ef4444) | 逾期警告 |
| 正常标记 | `--primary` (#6366f1) | 正常状态 |
| 熟练标记 | `--success` (#22c55e) | 高掌握度 |

### 2.3 字体规范

| 元素 | 字号 | 字重 | 颜色 |
|------|------|------|------|
| 卡片问题 | 17px | 600 | `--gray-900` |
| 答案解析 | 13.5px | 400 | `--gray-600` |
| 评分按钮 | 13px | 500 | `--gray-600` |
| 概览数字 | 32px | 700 | 多色可选 |

## 三、接口联调设计

### 3.1 数据集合操作

#### 3.1.1 `review_cards` — 复习卡片

| 操作 | SDK 方法 | 接口编号 |
|------|---------|---------|
| **今日待复习** | `.where({ nextReview: <= today, mastery: > 0 })` | REV-001 |
| **某知识的卡片** | `.where({ knowledgeId })` | REV-002 |
| **SM-2 更新** | `.doc(id).update({ mastery, interval, nextReview, ... })` | REV-003 |
| **创建卡片** | `.add({ question, answer, knowledgeId })` | REV-004 |

#### 3.1.2 `review_history` — 复习记录

| 操作 | SDK 方法 | 接口编号 |
|------|---------|---------|
| **某卡片历史** | `.where({ cardId }).orderBy('reviewedAt', 'desc')` | REV-005 |
| **写入记录** | `.add({ cardId, quality, interval, mastery })` | REV-006 |

#### 3.1.3 `knowledge_items` — 知识条目（关联查询）

| 操作 | SDK 方法 | 接口编号 |
|------|---------|---------|
| **查询关联知识** | `.where({ categoryId })` | REV-007 |
