## 一、技术架构说明

### 1.1 调用架构

| 类别 | 调用方式 | 接口形式 | 说明 |
|------|----------|----------|------|
| 数据库操作 | CloudBase JS SDK | `db.collection('xxx').xxx()` | 核心 CRUD，前端直连云数据库 |
| AI 调用 | CloudBase 云函数代理 | `app.callFunction('ai-proxy', data)` | AI Key 安全调用 |
| 认证 | CloudBase Auth SDK | `auth.signInWithEmailAndPassword()` | 邮箱/匿名登录 |

### 1.2 接口编号规范

| 接口类型 | 编号前缀 | 说明 | 前端调用标识 |
|----------|:--------:|------|:-----------:|
| 数据库读取 | `DB-R-xxx` | `db.collection().get()` | ✅ |
| 数据库写入 | `DB-W-xxx` | `db.collection().add()` | ✅ |
| 数据库更新 | `DB-U-xxx` | `db.collection().doc().update()` | ✅ |
| 数据库删除 | `DB-D-xxx` | `db.collection().doc().remove()` | ✅ |
| AI 调用 | `AI-xxx` | `app.callFunction('ai-proxy')` | ✅ |
| 聚合查询 | `AGG-xxx` | 多集合聚合查询 | ✅ |
| 云函数 | `CF-xxx` | 其他云函数调用 | ✅ |

| 编号 | 接口名称 | 删除原因 |
|:--:|---------|----------|
