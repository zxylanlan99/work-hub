## 二、用户故事

### 2.1 学习计划模块

#### US-001 目标创建
```
作为 学习者
我希望 能够创建学习目标
以便 明确我要学习什么

验收标准：
- [ ] 可以输入目标标题和描述
- [ ] 可以选择目标时间范围
- [ ] 目标保存后出现在目标列表
- [ ] 创建时可以选择目标分类（领域）
```

#### US-002 AI辅助目标拆解
```
作为 学习者
我希望 AI能帮我将大目标拆解为可执行的步骤
以便 我不知道从何下手时获得引导

验收标准：
- [ ] 输入模糊目标后，AI生成3-5个里程碑
- [ ] 每个里程碑包含标题、描述、预计完成时间
- [ ] 我可以编辑、删除或重排AI生成的里程碑
- [ ] 拆解结果保存为可执行的计划
```

#### US-003 里程碑管理
```
作为 学习者
我希望 能够管理计划中的里程碑
以便 跟踪大阶段进度

验收标准：
- [ ] 可以查看目标下的所有里程碑
- [ ] 可以添加、编辑、删除里程碑
- [ ] 里程碑显示完成状态
- [ ] 里程碑可以拖拽调整顺序
```

#### US-004 任务CRUD
```
作为 学习者
我希望 能够在里程碑下创建和管理任务
以便 细化具体的学习行动

验收标准：
- [ ] 可以为里程碑添加任务
- [ ] 任务包含标题、描述、截止日期、预计耗时
- [ ] 可以编辑和删除任务
- [ ] 任务支持优先级标记（高/中/低）
```

#### US-005 任务状态流转
```
作为 学习者
我希望 能够标记任务完成状态
以便 看到学习进展

验收标准：
- [ ] 任务状态：未开始/进行中/已完成
- [ ] 点击即可切换状态
- [ ] 已完成任务显示完成时间
- [ ] 已完成的任务可以重新打开
```

#### US-006 拖拽排序
```
作为 学习者
我希望 能够拖拽调整任务和里程碑顺序
以便 灵活调整计划

验收标准：
- [ ] 支持同层级拖拽排序
- [ ] 支持跨里程碑拖拽任务
- [ ] 拖拽有视觉反馈
- [ ] 排序结果自动保存
```

#### US-007 弹性排期
```
作为 学习者
我希望 调整一个任务的截止日期时，后续任务自动顺延
以便 计划变更时不用手动调整所有任务

验收标准：
- [ ] 修改任务截止日期时，系统提示"是否顺延后续任务"
- [ ] 选择"是"后，后续未完成任务自动顺延相同天数
- [ ] 已完成的任务不受影响
- [ ] 可以预览顺延结果再确认
```

#### US-008 进度仪表盘
```
作为 学习者
我希望 在首页看到学习进度总览
以便 一目了然当前状态

验收标准：
- [ ] 显示进行中的目标数量
- [ ] 显示今日待完成任务数
- [ ] 显示本周完成率（进度条）
- [ ] 显示本月学习时长统计
- [ ] 显示最近活跃的知识主题
```

#### US-009 日历视图
```
作为 学习者
我希望 在日历上看到任务分布
以便 直观了解时间安排

验收标准：
- [ ] 月视图显示所有任务
- [ ] 任务按优先级显示不同颜色
- [ - [ ] 点击日期可查看当天任务
- [ ] 支持拖拽任务到不同日期
```

#### US-010 看板视图
```
作为 学习者
我希望 以看板方式管理任务
以便 像Trello一样直观操作

验收标准：
- [ ] 三列看板：未开始/进行中/已完成
- [ ] 任务卡片显示标题、截止日期、优先级
- [ ] 支持拖拽任务到不同列
- [ ] 支持按里程碑筛选看板
```

#### US-011 主题锁定
```
作为 学习者
我希望 开启一个主题后，系统帮我屏蔽无关内容
以便 防止我贪多同时开多个坑

验收标准：
- [ ] 可以在目标/主题上点击"锁定"
- [ ] 锁定后首页只显示该主题相关内容
- [ ] 知识库默认筛选为该主题
- [ ] 有显眼的"主题锁定中"提示和解锁按钮
- [ ] 每天首次打开时显示当前锁定主题和进度
```

#### US-012 时间感知推荐
```
作为 学习者
我希望 系统根据当前时间推荐适合的学习内容
以便 碎片时间也能有效利用

验收标准：
- [ ] 晚上8点后推荐"深度模式"内容（1-2小时）
- [ ] 工作间隙推荐"碎片模式"内容（5-15分钟）
- [ ] 首次打开APP时推荐"暖身模式"（3分钟）
- [ ] 推荐内容基于当前锁定主题
```

#### US-013 学习模式切换
```
作为 学习者
我希望 主动选择不同的学习模式
以便 匹配我当前的状态

验收标准：
- [ ] 三种模式按钮：暖身/碎片/深度
- [ ] 暖身模式：显示一个复习卡片或知识点问答
- [ ] 碎片模式：显示短文章/摘要/闪卡
- [ ] 深度模式：显示完整知识条目或长文阅读
- [ ] 模式切换即时生效
```

#### US-014 每日日志
```
作为 学习者
我希望 记录每天的学习情况
以便 回顾和追踪学习习惯

验收标准：
- [ ] 每天自动生成日志条目
- [ ] 自动统计今日学习时长、完成任务数
- [ ] 可以手动填写心情和备注
- [ ] 支持查看历史日志日历
```

### 4.3 关键查询场景

```sql
-- 1. 首页仪表盘：用户活跃目标和今日任务
SELECT 
  g.*,
  COUNT(DISTINCT m.id) as milestoneCount,
  COUNT(DISTINCT t.id) as taskCount,
  COUNT(DISTINCT CASE WHEN t.status = 'completed' THEN t.id END) as completedTasks
FROM LearningGoal g
LEFT JOIN Milestone m ON m.goalId = g.id
LEFT JOIN Task t ON t.milestoneId = m.id
WHERE g.userId = ? AND g.status = 'active'
GROUP BY g.id;

-- 2. 今日待复习卡片
SELECT r.*, k.title, k.content, k.aiSummary
FROM ReviewCard r
JOIN KnowledgeItem k ON k.id = r.knowledgeId
WHERE r.userId = ? AND r.nextReviewDate <= date('now', 'start of day', '+1 day')
ORDER BY r.nextReviewDate ASC;

-- 3. 知识库搜索（FTS5）
SELECT k.*, c.name as categoryName
FROM KnowledgeItem k
JOIN KnowledgeItem_fts fts ON k.id = fts.rowid
JOIN Category c ON c.id = k.categoryId
WHERE KnowledgeItem_fts MATCH ? AND k.userId = ?
ORDER BY rank;

-- 4. 费用统计（本月）
SELECT 
  model,
  SUM(totalTokens) as totalTokens,
  SUM(totalCost) as totalCost,
  COUNT(*) as sessionCount
FROM ChatSession
WHERE userId = ? AND createdAt >= date('now', 'start of month')
GROUP BY model;

-- 5. 学习统计（近7天）
SELECT 
  date,
  studyMinutes,
  tasksCompleted
FROM DailyLog
WHERE userId = ? AND date >= date('now', '-7 days')
ORDER BY date ASC;
```
